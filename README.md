## C3PO
A Discord bot which is awesome.

Usage:
>Prefix: ```c!```
* ```c!ping```: Shows your ping
* ```c!hi/hello```: Greets you
* ```c!bye```: Says bye to you
* ```c!thanks {user}```: Say thanks to ```{user}```
* ```c!shoot {user}```: Shoots ```{user}```
* ```c!bruh```: Shows a image showing bruh moment
